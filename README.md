# wasm-prototyping
> WebAssembly prototyping

## Quickstart

### Install dependencies

Install [emscripten](https://emscripten.org/) and local toolchain
```
npm install
```

### Build Wasm target
```
bash build.sh
```

### Start HTTP server
```
npm run serve-build
```
Open http://localhost:8081 in your browser.
